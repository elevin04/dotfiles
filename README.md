# Dotfiles for my Lenovo Thinkpad x220

## General Info

- OS: Arch Linux

- wm: i3-gaps

- bar: polybar

- menu: rofi

- compositor: picom

- font: mononoki (terminal) and Font Awesome 5 (polybar icons)

- shell: bash (for now at least)

- editor: vim
